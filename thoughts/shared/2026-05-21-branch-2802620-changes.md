## Email Flow (SendMFAEmail)

```
POST api/sendmfaemail
  └─ EmailController.SendMFAEmail(MFAEmailRequest)
       └─ ManageEmail.SendMFAEmail(firstName, emailAddress, otpCode)
            ├─ GetEmailTemplate(AudienceType=5, PurposeType=5)
            │    └─ Enterprise.CommunicationEmailTemplate (DB)
            ├─ body.Replace("{FIRST NAME}", firstName)
            ├─ body.Replace("{AUTHENTICATION CODE}", otpCode)
            ├─ if IsUnifiedEmailEnabled=1 → SendEmailAsync(emailModel)
            │    └─ POST {UnifiedEmailBaseAddress}{UnifiedEmailEndPoint}
            └─ elif IsSendGridEnabled=1 → SendGridEmail(sendGridEmail)
                 └─ POST {SendGridApiEndPoint}{SendGridSendEmailEndPoint}
```

**Note (DEV environment):** UnifiedEmail's DEV SMTP relay (`elkdev.realpage.com`) only accepts whitelisted email domains. Use a `@realpage.com` login email for testing, or enable `IsSendGridEnabled=1` with `IsUnifiedEmailEnabled=0` to route through SendGrid instead.

## Template Placeholders

| Placeholder | Value |
|-------------|-------|
| `{FIRST NAME}` | Recipient's first name (from user record) |
| `{AUTHENTICATION CODE}` | 6-digit OTP code generated by unified-login-core |
| `{IMAGES}` | Not used in MFA template (present in other templates) |
