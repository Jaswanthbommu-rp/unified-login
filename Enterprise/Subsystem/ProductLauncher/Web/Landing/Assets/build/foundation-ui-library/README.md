# Foundation UI Library

## Usage
To use this module, simply include the following in your `packages.json` file.

```json
"dependencies" : {
  "foundation-ui-library" : "github:realpage-property-os/foundation-ui-library"
}
```

Files are also available on a CDN at the minor and patch level.  You can access the contents of `dist` at:

 * https://cdn.realpage.com/foundation-ui-library/vX.X
 * https://cdn.realpage.com/foundation-ui-library/vX.X.X
